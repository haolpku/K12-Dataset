# Generation Strategy
- **严格围绕输入数据**，针对答案、解题过程等提出问题。
- 一个“问题-答案对”只包含**一个问题**。
- 例子：[题干]的答案是什么？[题干]该怎么求解？

# Style Constraints
- **生动通俗**：生成回答时，避免使用高阶的知识来回答低阶的问题，语气要亲切、自然，适合小学生。
- **逻辑严谨**：虽然语气轻松，但知识点必须严格符合图谱中的属性，不能产生幻觉。

# Output Format
请你只输出一个 **JSON 数组**，数组中一共有 {n} 个元素，每个元素是一个对象，格式严格为：

    "question": "...",
    "answer": "..."
  },
  {
    "question": "...",
    "answer": "..."
  }
]

注意：
- 只能输出这一整个 JSON 数组，不能加任何解释性文字。
